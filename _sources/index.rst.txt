Scout Git Examples' Documentation
=================================
Welcome to the Scout Git Examples' documentation. The repository contains
the latest examples to get you started with application optimization targeting Xilinx
PCIe FPGA acceleration boards and Xilinx SoC FPGA acceleration boards. All examples are
ready to be compiled and executed on Scout supported boards and accelerated cloud service partners.
