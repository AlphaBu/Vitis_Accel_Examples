Prerequisites
=============

Vitis Accel Examples assumes that user is familiar with basic Vitis Environment, Setup, Programming and Debugging Flow. If not, it is recommended to cover these topics from Vitis Tutorials and User Guides:

    * `Vitis Accel Examples Tutorials`_
    * `Vitis Development Environment`_ 
    * `Vitis Programming Guide`_ 
    * `Vitis Debugging Guide`_ 

.. _Vitis Accel Examples Tutorials: https://github.com/Xilinx/SDAccel-Tutorials/blob/master/README.md
.. _Vitis Development Environment: https://www.xilinx.com/support/documentation/sw_manuals/xilinx2018_3/ug1023-sdaccel-user-guide.pdf
.. _Vitis Programming Guide: https://www.xilinx.com/support/documentation/sw_manuals/xilinx2018_3/ug1277-sdaccel-programmers-guide.pdf
.. _Vitis Debugging Guide: https://www.xilinx.com/support/documentation/sw_manuals/xilinx2018_3/ug1281-sdaccel-debugging-guide.pdf
