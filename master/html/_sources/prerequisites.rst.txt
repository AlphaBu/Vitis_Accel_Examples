Prerequisites
=============

Vitis Accel Examples assumes that user is familiar with basic Vitis Environment, Setup, Programming and Debugging Flow. If not, it is recommended to cover this topic from Vitis User Guides:

    * `Vitis Application Acceleration Guide`_
